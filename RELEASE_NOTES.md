## 🚀 快速开始

### 1. 安装应用

1. 下载 `app-release.apk` 文件
2. 在 Android 设备上打开文件
3. 如果提示"不允许安装未知来源"，请前往：
   - **设置** → **安全** → **允许安装未知应用**
   - 选择您的文件管理器或浏览器，允许安装
4. 点击"安装"

### 2. 配置服务器

首次启动后，前往 **设置** 页面配置您的服务器：

#### 必需配置
- ✅ **TMDB API Key** - 获取电影/电视剧元数据
  - 注册地址: https://www.themoviedb.org/settings/api

#### 可选配置（根据需要）
- **Radarr** - 电影管理
  - 服务器地址: `http://your-server:7878`
  - API 密钥: 在 Radarr 设置中获取
  
- **Sonarr** - 电视剧管理
  - 服务器地址: `http://your-server:8989`
  - API 密钥: 在 Sonarr 设置中获取
  
- **Whisparr** - 成人内容管理
  - 服务器地址: `http://your-server:6969`
  - API 密钥: 在 Whisparr 设置中获取
  
- **StashDB API Key** - 成人场景数据库（可选）
  - 注册地址: https://stashdb.org

### 3. 开始使用

配置完成后，即可开始浏览和管理您的媒体库！

## 🆘 故障排除

### 应用无法连接到服务器
1. 检查服务器地址是否正确（包括 `http://` 或 `https://`）
2. 确认服务器正在运行
3. 检查 API 密钥是否正确
4. 确保设备可以访问服务器网络
5. 尝试在浏览器中访问服务器地址

### TMDB API 配额超限
- TMDB 免费 API 有请求限制
- 如遇到限流，请稍后重试
- 考虑使用 TMDB 代理服务

### 应用崩溃或无响应
1. 清除应用数据：**设置** → **应用** → **Yesseerr** → **清除数据**
2. 重新配置服务器信息
3. 确保 Android 系统已更新
4. 查看 GitHub Issues 是否有已知问题
